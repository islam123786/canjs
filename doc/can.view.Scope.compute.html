compute

scope.compute( key, [options] )

Parameters

  1. key {key}:

    A dot seperated path. Use "." if you have a property name that includes a dot.

  2. options {readOptions}:

    Options that configure how the key gets read.

  • returns :

    A compute that can get or set key.
