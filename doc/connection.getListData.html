getListData

Retrieves list data for a particular set.

function(set)

Returns a promise that resolves to the list data for a particular set.

Parameters

  1. set {Object}:

    A object that represents the set of data needed to be loaded.

  • returns :

    A promise that resolves to the ListData format.

Use

Extensions like can-connect/data/url/url make it easy to implement getListData, but it can be as simple as:

var connection = connect([],{
  getListData: function(set){
    return $.get(set)
  }
})
