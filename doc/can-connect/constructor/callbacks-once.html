can-connect/constructor/callbacks-once

Prevents unecessary calls to the instance callback methods.

constructorCallbacksOnce(baseBehavior)

Prevents duplicate calls to the instance callback methods by tracking the last data the methods were called with. If called with the same data again, it does not call the base behavior's instance callback.
