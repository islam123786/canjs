render

Mustache.render(partial, context)

Parameters

  1. partial {Object}:
  2. scope {can.view.Scope}:

Mustache.render is a helper method that calls into can.view.render passing the partial and the context object.

Its purpose is to determine if the partial object being passed represents a template like:

partial === "movember.mustache"

or if the partial is a variable name that represents a partial on the context object such as:

context[partial] === "movember.mustache"
